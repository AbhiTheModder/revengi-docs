# Welcome to RevEngiBot Docs

RevEngiBot is a powerful Telegram bot tailored for reverse engineers and developers. It combines an array of essential tools designed to streamline workflows and simplify complex tasks such as Smali conversion, APK analysis, Java/Smali decompilation, ARM conversion, and much more.

## Getting Started

To start using RevEngiBot, visit [RevEngiBot on Telegram](https://t.me/RevEngiBot) and explore a suite of tools that can significantly enhance your reverse engineering projects.

## Commands Overview

Below is a complete list of commands you can use in the bot to interact with its various features:

<Callout type="info">**TIP:** Click on any command to see its detailed usage.</Callout>

| Command                                            | Description                                                                                     |
| -------------------------------------------------- | ----------------------------------------------------------------------------------------------- |
| [`/start`](./docs/commands/#start)                 | View the bot's start page.                                                                      |
| [`/help`](./docs/commands/#help)                   | Access the help section.                                                                        |
| [`/aab2apk`](./docs/commands/#aab-to-apk)          | Convert Android App Bundles (AAB) to APK.                                                       |
| [`/apk`](./docs/commands/#apk)                     | Analyze APK files, decompile components, extract resources, and more.                           |
| [`/apkid` ](./docs/commands/#apkid)                | Identify compilers, packers, obfuscators, and tools used in an APK.                             |
| [`/apkprotect`](./docs/commands/#apkprotect)       | Protect APK files against unauthorized modification.                                            |
| [`/apksign`](./docs/commands/#apksign)             | Sign APK files with multiple signing schemes (v1, v2, v3).                                      |
| [`/askai`](./docs/commands/#askai)                 | Use AI to ask technical questions.                                                              |
| [`/asm`](./docs/commands/#asm)                     | Convert assembly language to hexadecimal.                                                       |
| [`/disasm`](./docs/commands/#disasm)               | Convert hexadecimal back to assembly.                                                           |
| [`/base`](./docs/commands/#base)                   | Convert numbers between different bases (binary, hexadecimal, decimal, etc.).                   |
| [`/blutter`](./docs/commands/#blutter)             | Analyze Flutter APKs and `.so` files using Blutter integration.                                 |
| [`/credits`](./docs/commands/#credits)             | See third-party libraries used in the bot.                                                      |
| [`/cff`](./docs/commands/#cff)                     | Obfuscation(control flow flattening) dex.                                                       |
| [`/cocos2d`](./docs/commands/#cocos2d)             | Decrypt cocos2d encrypted js files.                                                             |
| [`/dex2c`](./docs/commands/#dex2c)                 | Translate dalvikvm code into native code.                                                       |
| [`/dex2java`](./docs/commands/#dex2java)           | Convert Dex files to Java source code. (Supports Multiple Deompiler options like jadx,cfr etc.) |
| [`/dex2jar`](./docs/commands/#dex2jar)             | Convert Dex files to JAR format.                                                                |
| [`/dexrepair`](./docs/commands/#dex-repair)        | Repair and optimize Dex files.                                                                  |
| `/flutter_sub`                                     | Subcribe to Flutter release notifications.                                                      |
| [`/frida_compile`](./docs/commands/#frida_compile) | Compile & Obfuscate JavaScript Frida scripts to Frida-compatible bytecode format.               |
| [`/hash`](./docs/commands/#hash)                   | Generate hash values from text or files (supports multiple algorithms).                         |
| [`/java2smali`](./docs/commands/#java2smali)       | Convert Java files to Smali code.                                                               |
| [`/jni_info`](./docs/commands/#jni_info)           | Find JNI function signatures in APK.                                                            |
| [`/mthook`](./docs/commands/#mthook)               | Generate MT Enhanced Hook.                                                                      |
| [`/pairip`](./docs/commands/#pairip)               | Patch PairIP in Split APKs.                                                                     |
| `/premium`                                         | Learn about RevEngiBot Premium and support development.                                         |
| [`/regex`](./docs/commands/#regex)                 | Convert Smali instructions to regular expressions.                                              |
| [`/s2f`](./docs/commands/#s2f)                     | Generate Frida hooks directly from .smali files, speeding up setup for dynamic analysis.        |
| [`/scan`](./docs/commands/#scan)                   | Perform APK security scans using integrated tools.                                              |
| [`/smali`](./docs/commands/#smali)                 | Query detailed Smali grammar and opcodes.                                                       |
| [`/smali2java`](./docs/commands/#java2smali)       | Convert Smali code to Java.                                                                     |
| [`ssl_patch`](./docs/commands/#ssl_patch)          | Patch SSL pinning in APKs.                                                                      |
| [`/toapk`](./docs/commands/#toapk)                 | Convert XAPK, APKM, or APKS files to APK.                                                       |
| [`/xml`](./docs/commands/#xml)                     | XML-related tools for decompiling, parsing, and editing.                                        |
